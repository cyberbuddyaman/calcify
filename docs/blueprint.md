# **App Name**: Calcify

## Core Features:

- Number Input: Display a number input field and a keypad for numeric entry.
- Display Screen: Show a display screen that presents the current input and results of calculations.
- Basic Arithmetic Operations: Render buttons for the four basic arithmetic operations (addition, subtraction, multiplication, division).
- Calculation Engine: Implement the logic to perform calculations based on user input and selected operations.
- Operation Chaining: Allow the calculator to chain operations together to complete more complex calculation chains.
- Trigonometric Functions: Add button for basic trigonometric functions such as sine, cosine and tangent.
- AI Equation Simplifier: Use an AI powered tool that offers equation simplifications, if any.

## Style Guidelines:

- Primary color: Midnight blue (#2C3E50) to create a professional, focused atmosphere.
- Background color: Soft gray (#F0F4F8), providing a clean, neutral backdrop that reduces screen glare and ensures readability.
- Accent color: Teal (#3498DB), offering a bright, analogous contrast to the primary, used to highlight interactive elements like buttons.
- Body and headline font: 'Inter' sans-serif for a modern, neutral feel, ideal for both display and keypad.
- Simple, geometric icons for operation buttons.
- Use a grid-based layout for keypad arrangement.
- Subtle animations for button presses and result displays.
